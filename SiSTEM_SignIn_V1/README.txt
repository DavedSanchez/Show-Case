SiSTEM Sign In:
	Graphic based sign in for memebers of the SiSTEM Club at UCF
	
	Logic:
		Loads in and assorts members by name
		Checks for duplicates and matches via school ID
		Files are saved as .csv for easy ripping, loading, reading, and writing
		When master .csv is populated, signing-in members are cross ref to ensure they are actual members
	Design:
		Admin is able to chage logo and background colour
		Admin is able to load new members to master .csv
